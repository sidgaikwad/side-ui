# Cards

Container components for organizing and highlighting content in terminal UIs.

## Basic Card

Simple card with title, subtitle, and content area.

```tsx title="BasicCard"
import { BasicCard } from 'siddcn';

<BasicCard title="Card Title" subtitle="Subtitle">
  <Text>Card content here</Text>
</BasicCard>
```

### Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `title` | `string` | - | Card title |
| `subtitle` | `string` | - | Optional subtitle |
| `children` | `ReactNode` | - | Card content |
| `width` | `number` | `40` | Card width |

## Usage Example

```tsx title="Dashboard Cards"
import { Box } from 'ink';
import { InfoCard, SuccessCard, WarningCard } from 'siddcn';

function Dashboard() {
  return (
    <Box flexDirection="row" gap={2}>
      <SuccessCard title="Active">
        <Text>12 services running</Text>
      </SuccessCard>
      <WarningCard title="Alerts">
        <Text>3 warnings</Text>
      </WarningCard>
      <InfoCard title="Info">
        <Text>System healthy</Text>
      </InfoCard>
    </Box>
  );
}
```
