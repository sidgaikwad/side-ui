## Usage Pattern

All components follow a consistent usage pattern:

```tsx title="Example"
import { ComponentName } from 'siddcn';

function App() {
  return (
    <ComponentName
      prop1="value"
      prop2={42}
      onEvent={() => console.log('Event!')}
    />
  );
}
```

## Theming

All components automatically support theming:

```tsx title="Themed Component"
import { ThemeProvider, SimpleButton } from 'siddcn';

function App() {
  return (
    <ThemeProvider theme="tokyo-night">
      <SimpleButton label="Themed Button" />
    </ThemeProvider>
  );
}
```

See the [Themes](/docs/themes) page for more information.
