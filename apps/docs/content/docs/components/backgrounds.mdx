# Backgrounds

Animated background effects to enhance your terminal application's visual appeal.

## Twinkling Stars

Static starfield with twinkling brightness animation.

```tsx title="TwinklingStars"
import { TwinklingStars } from 'siddcn';

<TwinklingStars width={60} density={0.15} />
```

### Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `width` | `number` | `60` | Width of the starfield |
| `density` | `number` | `0.15` | Star density (0-1) |

## Matrix Rain

Matrix-style falling characters animation.

```tsx title="MatrixRain"
import { MatrixRain } from 'siddcn';

<MatrixRain width={60} height={5} />
```

### Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `width` | `number` | `60` | Width of the rain effect |
| `height` | `number` | `5` | Height of the rain effect |

## Animated Stars

Full animated starfield with falling stars.

```tsx title="AnimatedStars"
import { AnimatedStars } from 'siddcn';

<AnimatedStars 
  width={80} 
  height={24} 
  starCount={30}
  fps={10}
/>
```

### Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `width` | `number` | `80` | Width of the background |
| `height` | `number` | `24` | Height of the background |
| `starCount` | `number` | `30` | Number of stars |
| `fps` | `number` | `10` | Animation frame rate |

## Diagonal Falling Stars

Shooting star trails moving diagonally across the screen.

```tsx title="DiagonalFallingStars"
import { DiagonalFallingStars } from 'siddcn';

<DiagonalFallingStars 
  width={100} 
  height={30}
  starCount={15}
  fps={12}
/>
```

## Static Patterns

For performance-sensitive applications, use static backgrounds:

```tsx title="StaticDiagonalPattern"
import { StaticDiagonalPattern } from 'siddcn';

<StaticDiagonalPattern 
  width={100} 
  height={30}
  density={0.08}
/>
```

## Usage with Layout

```tsx title="Full Screen Background"
import { Box } from 'ink';
import { FullScreenBackground } from 'siddcn';

function App() {
  return (
    <FullScreenBackground width={100} height={30} starCount={12}>
      <YourContent />
    </FullScreenBackground>
  );
}
```
